Morse Learner

How to play

With this cool gaming app you can start easily with lernen the Morse-alphabet and -numbers.

There are currently 2 different game modes, the Easy Game and the Advanced Game.

Easy Game

Given a random morse-code, each round you have to choose the correct Character or Number representation of the code , out of 4 different Options.

Advanced Game

In each round you have to identify a random morse-code.

When pressing the button, you can feel the vibration as often as you want.
